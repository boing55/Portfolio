# Project 1: IPL Player Analysis

## Description
This project analyzes IPL players' prices, roles, and teams. 
It includes data cleaning, analysis, and visualization using SQL Server, R, and Power BI.

## Objective
- Understand team spending patterns
- Identify top players by role and team
- Visualize insights using Power BI dashboards

## Folder Structure
- `Data/` : contains all datasets used
- `SQL_Server/` : SQL queries and scripts
- `R/` : R scripts for analysis
- `PowerBI/` : Power BI reports
